# 🚀 SaaS Analytics Dashboard

A modern and premium SaaS dashboard built using React, Tailwind CSS, and Recharts.  
Designed with a focus on clean UI, smooth user experience, and real-world dashboard features.

---

## ✨ Features

- 📊 Interactive Charts (Line, Bar, Pie)
- 🌙 Dark Premium UI (Glass + Gradient Design)
- 📱 Fully Responsive (Mobile, Tablet, Desktop)
- ⚡ Smooth Animations & Hover Effects
- 📈 Data Visualization Dashboard
- 🔍 Search UI & Navigation System

---

## 🛠️ Tech Stack

- React (Vite)
- Tailwind CSS (v4)
- Recharts (Charts & Data Visualization)
- Lucide Icons
- Framer Motion (for animations)
